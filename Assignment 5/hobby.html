What is a Raspberry Pi?

The raspberry pi is a small single board computer that costs about $35. Raspberry Pi's initial purpose was to help children and others learn to program. Hobbyists have used pi's for all kinds of projects and there is a community that uses the pi to make a retro-gaming console. There is an image referred to as "RetroPie" that uses rasbian-os, emulationstation, and retroarch to make an easy to use and extremely customizable emulation machine.

Some of RetroPie's most popular supported systems include:

  • Atari 2600
  • Atari 7800
  • Atari Lynx
  • GCE Vectrex
  • NEC TurboGrafx-16
  • Nintendo 64
  • Nintendo Entertainment System
  • Super NES
  • Nintendo Game Boy>
  • Nintendo Game Boy Color
  • Nintendo Game Boy Advance
  • Nintendo Virtual Boy
  • Sega 32X
  • Sega CD
  • Sega Master System
  • Sega Genesis
  • Sega Game Gear
  • Sega Saturn
  • SNK Neo Geo
  • SNK Neo Geo Pocket Color
  • Sony PlayStation
  • Sony PSP

More info on RetroPie can be found here: RetroPie.org
